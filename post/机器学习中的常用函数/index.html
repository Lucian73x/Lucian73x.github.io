机器学习中的常用函数
2026-02-19

logistic 函数

Definition (logistic 函数)
\[ f(x) = \frac{1}{1+e^{-x}} \]

定义域和值域

  • 定义域: \((-\infty, +\infty)\)
  • 值域: \((0, 1)\)

图像

导数

Solution.

计算有

\[ f'(x) = \frac{e^{-x}}{(1+e^{-x})^2} = \frac{1}{1+e^{-x}}\frac{e^{-x}}{1+e^{-x}} = \frac{1}{1+e^{-x}}(1 - \frac{1}{1+e^{-x}}), \]

故有

\[ f'(x) = f(x)(1 - f(x)). \]

作用

在 logistic 回归和神经网络中会使用到, 可作为神经网络的激活函数.

ReLU 函数

定义

Definition (ReLU 函数)
\[ f(x) = \begin{cases} x,\ x \ge 0 \\ 0,\ x < 0 \end{cases} \]

定义域和值域

  • 定义域: \((-\infty, +\infty)\)
  • 值域: \([0, \infty)\)

图像

导数

ReLU 函数在原点处不可导.

Solution.
\[ f'(x) = \begin{cases} 1, &x > 0 \\ 0, &x < 0 \end{cases} \]

作用

ReLU 函数常用做神经网络的激活函数.

softplus 函数

定义

softplus 函数可以看作是 ReLU 函数的光滑近似.

Definition (softplus 函数)
\[ f(x) = \ln{(1 + e^x)} \]

定义域和值域

  • 定义域: \((-\infty, +\infty)\)
  • 值域: \((0, +\infty)\)

图像

导数

softplus 的导数为 logistic 函数.

Solution.
\[ f'(x) = \frac{e^x}{1 + e^x} = \frac{1}{1 + e^{-x}} \]

绝对值函数

定义

Definition (绝对值函数)
\[ f(x) = |x| = \begin{cases} x, &x \ge 0 \\ -x, &x < 0 \end{cases} \]

定义域和值域

  • 定义域: \((-\infty, +\infty)\)
  • 值域: \([0, +\infty)\)

图像

导数

绝对值函数在原点不可导.

Solution.
\[ f'(x) = \begin{cases} 1, &x > 0 \\ -1, &x < 0 \end{cases} \]

作用

绝对值函数常用于构造机器学习算法训练目标函数中的正则化项.

sgn 函数

定义

Definition (符号函数)
\[ f(x) = \begin{cases} 1, &x \ge 0 \\ -1, &x < 0 \end{cases} \]

定义域和值域

  • 定义域: \((-\infty, +\infty)\)
  • 值域: \([-1, 1]\)

图像

导数

sgn 函数在原点不可导, 其他点导数均为 0.

Solution.
\[ f'(x) = \begin{cases} 0, &x \neq 0 \\ 不可导, &x = 0 \end{cases} \]

作用

sgn 函数常用于分类器的预测函数, 表示二值化的分类结果. 在支持向量机, logistic 回归中都被使用. 函数值为 1 时表示分类结果为正样本, -1 时为负样本.

注

ReLu, 绝对值函数和 sgn 函数都是几乎处处可导的函数.

人工神经网络中的激活函数需要满足几乎处处可导的条件, 以保证在训练时能够使用梯度下降法求解函数的极值.
