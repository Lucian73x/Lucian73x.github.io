函数图像

以 logistic 函数为例.

有两种风格, 一种是标准的带坐标轴的图像, 另一种是简单的图像.

# 导入使用的库
import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-6, 6, 500) # 在 -6 到 6 之间均匀生成500个数, 用作 x 的值
y = 1 / (1 + np.exp(-x)) # y 对应的值

# 1. 标准的图像
# 创建画布和坐标系
fig, ax = plt.subplots()
ax.plot(x, y) # 用 x 和 y 的数据画出函数曲线
ax.set_title("Logistic Function") # 图像标题
ax.grid(False) # 不显示背景表格
ax.spines["left"].set_position("zero") # y 轴
ax.spines["bottom"].set_position("zero") # x 轴
# 隐藏右边和下边的边框线
ax.spines["right"].set_visible(False)
ax.spines["top"].set_visible(False)
# 设置坐标系的箭头
ax.plot(1, 0, ">k", transform=ax.get_yaxis_transform(), clip_on=False)
ax.plot(0, 1, "^k", transform=ax.get_xaxis_transform(), clip_on=False)
# 设置坐标系的标识
ax.set_xlabel("X", loc="right")
ax.set_ylabel("Y", loc="top", rotation=0)

# 2. 简单的图像
plt.plot(x, y)

# 显示图像
plt.show()
