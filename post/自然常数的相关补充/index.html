单调数列单调性的证明 - P5

设

\[ e_n = \left(1 + \frac{1}{n}\right)^n, a_n = \left(1 + \frac{1}{n}\right)^{n+1}, \]

有 \(\{e_n\}\) 单调递增, \(\{a_n\}\) 单调递减.

书中只证明了关于 \(\{e_n\}\) 的结果, 使用的方法为二项式定理展开之后放缩.

下面将使用 \(AM\text{-}GM\) 不等式来证明相关结论.

\(\{e_n\}\) 单调递增

Proof.

由 \(AM\text{-}GM\)不等式知

\[ e_n = \left(1 + \frac{1}{n}\right)^n = 1\cdot\left(1 + \frac{1}{n}\right)^n \le \left(\frac{1 + n\cdot\left(1 + 1/n\right)}{n+1}\right)^{n+1} = \left(1 + \frac{1}{n+1}\right)^{n+1} = e_{n+1}, \]

从而 \(\{e_n\}\) 单调递增.

\(\{a_n\}\) 单调递减

Proof.

由 \(AM\text{-}GM\)不等式知

\[ \frac{1}{a_n} = \left(\frac{n}{n+1}\right)^{n+1} = 1\cdot\left(\frac{n}{n+1}\right)^{n+1} \le \left(\frac{1+n}{n+2}\right)^{n+2} = \frac{1}{a_{n+1}}, \]

即

\[ a_{n+1} \le a_n, \]

从而 \(\{a_n\}\) 单调递减.

\(\displaystyle\lim_{x\to+\infty}\left(1+1/x\right)^x=e\) 的证明 - P9

书中说可以将 (1.3) 式推广到函数极限的情况, 即 (1.8) 式, 但没有给出证明.

下面补充其证明.

Proof.

由取整函数知

\[ [x] \le x < [x]+1, \]

从而有

\[ \left(1+\frac{1}{[x]+1}\right)^{[x]} \le \left(1+\frac{1}{x}\right)^{[x]} \le \left(1 + \frac{1}{x}\right)^x \le \left(1+\frac{1}{[x]}\right)^x \le \left(1+\frac{1}{[x]}\right)^{[x]+1}, \]

又有

\[ \begin{aligned} &\lim_{x\to+\infty}\left(1+\frac{1}{[x]}\right)^{[x]+1} = \lim_{[x]\to+\infty}\left(1+\frac{1}{[x]}\right)^{[x]}\lim_{[x]\to+\infty}\left(1+\frac{1}{[x]}\right) = e,\\ &\lim_{x\to+\infty}\left(1+\frac{1}{[x]}\right)^{[x]} = \lim_{[x]\to+\infty}\left(1+\frac{1}{[x]+1}\right)^{[x]+1}\lim_{[x]\to+\infty}\left(1+\frac{1}{[x]+1}\right)^{-1} = e, \end{aligned} \]

由极限的迫敛性知

\[ \lim_{x\to+\infty}\left(1+\frac{1}{x}\right)^{x} = e. \]
